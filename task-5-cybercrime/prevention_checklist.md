# Stay Safe Online: Prevention Checklist
**By:** Aditya Prakash | **Reg ID:** 25MIP10104

## Actionable Tips for Students
- [ ] **UPI Safety:** Never enter a PIN to receive money.
- [ ] **QR Code Caution:** Do not scan QR codes sent by strangers to "receive" payments.
- [ ] **Enable MFA:** Secure GitHub, LinkedIn, and University accounts with 2FA.
- [ ] **Secure Coding:** Only use official libraries for Python/C++ projects.
- [ ] **Network Security:** Avoid using public Wi-Fi for banking or VTOP.
- [ ] **Privacy Audit:** Revoke unnecessary app permissions (Contacts/SMS).
- [ ] **Link Verification:** Always check for HTTPS and valid domains before login.
- [ ] **System Updates:** Keep OS and security software patched.
